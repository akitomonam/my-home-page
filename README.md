# モダンなプロフィールカード

React + TypeScript + Vite で構築された、モダンなプロフィールカードのデモプロジェクトです。

## 特徴

- 🎨 shadcn/ui を使用したモダンなデザイン
- 🌙 ダークモード対応
- 🎭 アニメーション効果
- 📱 レスポンシブデザイン
- 🛠 TypeScript による型安全性

## 技術スタック

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Lucide Icons

## 始め方
1. リポジトリをクローン:
    ```bash
    git clone https://github.com/your-username/modern-profile-card.git
    cd modern-profile-card
    ```

2. 依存関係をインストール:
    ```bash
    yarn install
    ```

3. 開発サーバーを起動:
    ```bash
    yarn dev
    ```

4. ビルド:
    ```bash
    yarn build
    ```
